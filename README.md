# ALTARIX — Aerospace Standard Atmosphere Engine

ALTARIX is a physics-driven aerospace web platform implementing the International Standard Atmosphere (ISA) model up to 86 km altitude.

It provides precise atmospheric property computation for:
• Flight analysis
• CFD preprocessing
• Aerospace simulation workflows

Key Features:
- Layer-based ISA modeling
- Temperature, pressure, density calculations
- Speed of sound and viscosity modeling
- Export-ready engineering outputs

Tech Stack:
React + TypeScript + Vite + TailwindCSS
