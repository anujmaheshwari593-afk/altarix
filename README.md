# ALTARIX — Aerospace Standard Atmosphere Engine

ALTARIX is a physics-driven web platform for calculating International Standard Atmosphere (ISA) properties up to 86 km altitude.

Features:
• Temperature, pressure, density modeling
• Speed of sound & viscosity computation
• Layer-based ISA segmentation
• Aerospace-ready outputs for CFD & flight analysis

Tech Stack:
React + TypeScript
Vite
TailwindCSS
